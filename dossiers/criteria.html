Модель 01
One-shot / OTP

Один продаж — один чек. CAC покривається з першої покупки. Фокус на маржі, wow-факторі і конвертованості трафіку.

Маржа landed
COG + доставка + мита. ≥3× — мінімум для прибуткового Media buying.
0 / 10 pts
AOV potential
Чи можна взяти ≥$50 з одного замовлення (bundle, upsell, ціна продукту)?
0 / 8 pts
Meta policy risk
Supplements/health = high. Beauty/grooming = med. Lifestyle/food = low.
0 / 10 pts
FB Ad Library saturation
Dead = немає конкурентів (ніша не перевірена). Live = є гравці, але є місце. Hot = всі вже там.
0 / 7 pts
Wow-фактор для відео
1 = нічого цікавого на камері. 5 = сам по собі зупиняє скрол.

UGC-able

Shippability

Модель 02
Subscription / LTV

CAC → LTV → payback. Перший продаж може бути в нуль — монетизація на refill і retention. Інша математика, інші ризики.

Consumability cycle
Як часто клієнт природно закінчує і замовляє знову? Monthly = золото.
0 / 10 pts
Gross margin
≥70% — мінімум для того щоб CAC payback склався. Нижче — підписка не виправдана.
0 / 10 pts
First-order hook
Чи є trial offer щоб зняти об'єкцію при першій покупці? $1 trial / free+shipping / перша коробка зі знижкою.
0 / 8 pts
CAC payback
За скільки місяців підписки окупається CAC? ≤90 днів = 3 поставки max.
0 / 10 pts
LTV:CAC до M6
Projected LTV через 6 місяців проти CAC. ≥3:1 — мінімум для масштабу.
0 / 10 pts
Habit fit
Продукт природно вбудовується в денну/тижневу рутину? Низький habit fit = "забув і скасував".
0 / 10 pts
Regulatory risk
FDA labeling + COA для supplements. FTC click-to-cancel. EU subscription laws якщо плануєш туди.
0 / 8 pts
Supply reliability
Чи можеш тримати стабільний інвентар під auto-ship? 1 постачальник = ризик стопу підписки.
0 / 7 pts
Shippability
Кожна додаткова доставка в підписці — це hit по маржі. Flat-mailer = ідеально.
0 / 7 pts
Perceived quality
Клієнт не соромиться тримати на видному місці. Це впливає на retention напряму — шлак не залишають.
