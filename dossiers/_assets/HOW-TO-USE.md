# Як заводити новий бренд у дозьє

## Крок 1: Створи папку бренду

Скопіюй папку `.template/` і назви її коротким slug-ом бренду (одне слово, lowercase, без пробілів):

```
_assets/
  posturefix/         ← slug бренду
    creatives/        ← сюди .mp4, .mov, .webm — будь-який формат
    screenshots/      ← скріни ad card з FB Ad Library
    product/          ← скріни продуктової сторінки (опційно)
```

## Крок 2: Закинь файли

- **creatives/** — всі відео-креативи. Назви як завгодно (1.mp4, hook-ugc.mp4 — байдуже).
- **screenshots/** — повні скріни ad card з Ad Library: header брендa, дата запуску, CTA, текст ads. Один скрін = один креатив.
- **product/** — якщо хочеш зберегти скрін самої продуктової сторінки (необовʼязково, я і сам її прочитаю).

## Крок 3: Пиши мені

```
готово, бренд posturefix, продукт: https://posturefix.com/products/v2
```

Якщо в якомусь відео є voiceover без впалених субтитрів — додай одним рядком що говорять.

## Що відбувається далі

1. Я фетчу продуктову сторінку (якщо це funnel — проходжу до реального оффера)
2. Витягую кадри з кожного відео (ffmpeg, 8-10 ключових кадрів кожне)
3. Аналізую креативи — хук, концепція, чому працює
4. Генерую `dossiers/posturefix.html` з повним аналізом
5. Додаю карточку у `index.html` (gallery)
6. Скидаю тобі лінк

Папка `_assets/posturefix/` залишається — дозьє посилається на відео і скріни звідти через відносні шляхи.

## Слаги: як називати

Коротко, lowercase, без пробілів. Якщо бренд `Pet Hair Magic` — slug `pethair` або `pethairmagic`. Якщо два бренди в одній ніші — додавай суфікс: `posturefix-1`, `posturefix-2`. Slug використовується як назва файлу і URL — змінити потім можна, але краще сразу нормально.
